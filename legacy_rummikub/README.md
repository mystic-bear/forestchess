# Legacy Rummikub Isolation

These files were part of the earlier Rummikub codebase and are no longer used by the active Forest Chess runtime.

Isolated during Phase 5:

- `shared/rules-core.js`
- `worker/ai-base.js`
- `worker/ai-dispatcher.js`
- `worker/ai-levels.js`
- `worker/ai-utils.js`
- `worker/hint-engine.js`

They remain in the repository only for historical reference and should be excluded from current chess-focused full-code snapshots.
